Welcome to the Rich Bank Project

Description

This is a project to be able to enter a new customer signup and view, update their information from database.
This allows a user to create at least account or joined account with other customer (saving/checking).
This application allows a user to deposit, withdraw, and transfer money from their account.

User Story

As a user, I should be able to:

  •  login my existing credentials
  •  create at least one account
  •  deposit an account (use doubles, not ints)
  •  withdraw from an account (** overdrafting!)
  •  view of my account(s) (all displays must be in proper currency format)
  •  register a new user account the application. (must be secured with a password)

Suggested Bonus User Stories

As a user, I should:

  •  able to see the transaction history for account
  •  able to create multiple accounts per user (checking/savings)
  •  able to share a joint account with another user
  •  transfer money between accounts
